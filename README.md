> A new redesigned Cultural Thesaurus Website, built for Keshia Kola at Shesaurus Labs, LLC. React Front-end w/ Material UI, Express and Node.js backend running on a MongoDB instance.

## Prerequisites

- node 13.12.0
- npm 6.14.4

## Install

```sh
npm install
```

## Usage

```sh
npm run development
```
